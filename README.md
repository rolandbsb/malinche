# 🐻 Malinche: Cultura, Comercio y el Oso Pinolillo JARVIS O.S. 🌋🌾

¡Bienvenido a **Malinche**, la plataforma premium interactiva para explorar, aprender y comprar lo más selecto de la herencia nicaragüense! Desde exquisitas artesanías de barro de San Juan de Oriente de Masaya, café de altura matagalpino, hasta los tradicionales dulces de Diriomo. Enmarcado con un diseño visual ultra pulido y de alto contraste bajo un ambiente confortable.

Toda la plataforma está custodiada por nuestra mascota oficial autómata: el **Oso Pinolillo**, un tierno oso de anteojos del Volcán Mombacho que ahora cuenta con funciones de inteligencia artificial avanzada y automatización en caliente inspirada en **JARVIS** (de Iron Man).

---

## 🚀 Características Clave (¿Qué hace la app?)

### 1. Comercio Electrónico Sabor a Pinol 🏺☕🍬
*   **Catálogo Curado:** Navega fácilmente entre artesanías finas, repostería/cajetas típicas, y cafés de especialidad de fincas seleccionadas.
*   **Carrito de Compras Nica:** Añade productos, selecciona tu departamento, calcula envíos automáticos y simula órdenes seguras con opción de **Pago Contra Entrega**.

### 2. Faro Cultural e Histórico 💡
*   Aprende detalladamente sobre las técnicas milenarias de los artesanos indígenas de Masaya, la cadena de valor del café norteño, y la elaboración artesanal de cajetas tradicionales.

### 3. Trivia Desafío Pinolero (Aprende Jugando) 🧠🎶
*   Pon a prueba tu nicaraguanidad en un juego interactivo de trivia. Descubre curiosidades sobre leyendas típicas, volcanes, lagos y la poesía de nuestro insigne exponente Rubén Darío.

### 4. Consola de Ventas e Intermediación (KPIs Dinámicos) 📈📊
*   Visualiza gráficos interactivos de ventas, controla ingresos, calcula estimaciones de comisiones con tasas variables, y gestiona pedidos simulados autorizados.

### 5. Generación Dinámica de Documentos Oficiales (Facturas y Catálogos en PDF) 📂📄
*   **Exportación Certificada (PDF):** Genera e imprime informes financieros de ventas y recetas de despacho con diseño administrativo.
*   **Visor de Doble Modo Resistente a Bloqueos:** Cuenta con una interfaz avanzada que ofrece:
    *   **Visor Digital de Hojas (HTML Semántico):** Un panel de alta fidelidad que simula las páginas reales del catálogo, libre de restricciones de seguridad de iframe/sandbox, permitiendo consultar los detalles del catálogo filtrado de viaje y en cualquier pantalla (móvil o tablet).
    *   **Lector PDF de Archivo (Iframe):** Vista clásica del archivo PDF generado en vivo.
    *   **Impresión y Guardado Nativo:** Sistema integrado para disparar la caja de impresión del navegador y resguardar en formato PDF nativo o imprimir físicamente con un clic, evadiendo bloqueos de descargas.

### 6. 🤖 Oso Pinolillo JARVIS O.S. (Inteligencia de Acción)
El Oso Pinolillo es más que una mascota: es el corazón inteligente del aplicativo. Tiene tres modos de operación integrados en la pestaña **Asistente Inteligente**:

*   **Olfato de Oso (Asistente de Compras):** Explícale tus gustos, antojos o intereses coloquialmente. Usando **Gemini AI**, analizará el catálogo de inmediato para recomendarte el producto idóneo y te dará una explicación súper cariñosa con dialecto y modismos nicaragüenses (*"primito"*, *"tuani"*, *"de viaje"*).
*   **Oso Historiador (Curiosidades Nicas con Google Search Grounding):** Hazle preguntas de folklore nicaragüense, recetas de nacatamal, mitos coloniales (*El Cadejo, La Cegua, La Carreta Nagua*) o datos geográficos. Pinolillo busca en internet con **Google Search Grounding en tiempo real** para darte información 100% verídica.
*   **Centro de Mandos JARVIS (Terminal Interactiva & Automatización):** Ahora puedes darle órdenes directas como si fuera un sistema operativo de ciencia ficción:
    *   `volcan` o `erupcion` 🌋: Despacha un protocolo de simulación sismográfica Momotombo variando la tasa de comisión a un estado volcánico en los gráficos financieros.
    *   `marimba` o `sonar` 🎶: Ejecuta una melodía automatizada de marimba de arco que ilumina individualmente las teclas interactivas de hormigo.
    *   `ir a la tienda / quiz / admin...` 🧭: Pinolillo calcula las coordenadas DOM y te teletransporta volando con scroll suave y cinemático.
    *   `modo nocturno / claro` 🌙: Alterna instantáneamente el tema de color luminoso/oscuro de toda la interfaz.
    *   `comprar` o `añadir` 🛒: Agrega automáticamente el último producto sugerido al carrito de compra.

## 🛠️ Lenguajes de Programación y Tecnologías

Malinche está construida utilizando una arquitectura de vanguardia full-stack moderna y limpia, optimizada para ofrecer fluidez total, interactividad inmediata de 60fps y seguridad de secretos en el lado del servidor:

*   **Lenguajes de Programación:**
    *   **TypeScript (TSX):** Utilizado de forma generalizada en el desarrollo de la aplicación (frontend y backend) garantizando tipado estrico, menos errores lógicos y máxima fiabilidad.
    *   **JavaScript (ES6+):** En rutinas dinámicas, animaciones y eventos personalizados del navegador.
    *   **HTML5/CSS3:** Formato estructural de marcado semántico de alta accesibilidad.

*   **Frameworks y Librerías de Interfaz (Frontend):**
    *   **React 18:** Arquitectura basada en componentes funcionales declarativos, React Hooks (`useState`, `useEffect`, `useRef`) para una sincronización idónea de estados de compras, comandos e interacciones de usuario.
    *   **Vite:** Herramienta de compilación y empaquetado de última generación, súper veloz y optimizada para servir la aplicación cliente SPA sin retrasos.
    *   **Tailwind CSS (V4):** Para un diseño artesanal, sofisticado, receptivo y fluido. Configurado de manera global mediante `@import "tailwindcss"`.
    *   **motion/react (Framer Motion):** Orquesta transiciones fluidas, stagger de cargas, micro-interacciones lúdicas en el avatar del oso Pinolillo, menús expansivos y el HUD holográfico del Jarvis OS.
    *   **Lucide React:** Colección unificada de iconos vectoriales modernos y estilizados.

*   **Entorno de Servidor (Backend & Integraciones de IA):**
    *   **Node.js & Express.js:** En ruteo REST API (`server.ts`) para procesamiento seguro. Garantiza el aislamiento de claves de API (API Keys) y evita la exposición de credenciales al navegador.
    *   **@google/genai SDK:** SDK nativo y moderno de Google para interactuar de forma segura y eficiente con los modelos de frontera **Gemini 3.5 Flash**, aprovechando esquemas JSON estructurados y **Google Search tool** para Grounding en línea.

## 📦 Ejecución y Desarrollo Local

### Requisitos Previos:
1. Instalar [Node.js](https://nodejs.org) (v18 o superior recomendado) y npm.

### Instalación de dependencias:
```bash
npm install
```

### Ejecutar Servidor de Desarrollo Híbrido:
Este comando arranca el backend Express en el puerto estático `3000` con el adaptador Vite interconectado para desarrollo caluroso:
```bash
npm run dev
```

### Compilar para Producción:
Genera los activos minimizados de producción y compila la lógica del servidor en el directorio `/dist`:
```bash
npm run build
```

### Arrancar en Producción:
Inicia la aplicación compilada tal y como se desplegará en la nube segura:
```bash
npm run start
```
